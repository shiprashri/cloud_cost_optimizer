# Cloud Optimizer

Phase 1: Basic project skeleton.

## How to run backend
1. `cd backend`
2. create & activate venv: `python -m venv venv` and activate
3. `pip install -r requirements.txt`
4. `python app.py`

Backend runs at `http://127.0.0.1:5000`

## How to open frontend
Open `frontend/index.html` in your browser (or serve via a static server).
